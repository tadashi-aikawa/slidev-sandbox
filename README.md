# Slidev Sandbox

Slidevのサンドボックスプロジェクト。コピーして使える。

## インストール

```bash
bun ci
```

## 起動

```bash
bun dev
```

http://localhost:3030 にアクセス。

## ビルド

```bash
bun run build
```

確認は

```bash
bun vite preview
```

## 編集

Neovimを使っているなら [slidev-preview.nvim](https://github.com/tadashi-aikawa/slidev-preview.nvim) がオススメ。
